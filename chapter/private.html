隐私接口

隐私接口

注意：隐私接口需要进行签名验证。

获取账户ID

限速规则：/

HTTP请求

GET /user/get/accoun

响应示例

{
    "data": {
        "memberId": 11
    },
    "code": 0,
    "message": "SUCCESS",
    "total": null
}

查询个人账户信息

限速规则：/

HTTP请求

GET /user/account

参数

memberId 账户ID

响应示例

{
    "data": [{
        "memberId": 11,
        "coinId": "USDT",
        "balance": 3207100.00831661,
        "frozenBalance": 363567515.696024,
        "address": "0x70edb9e59161006d53e2b5af0fcd9f45aa0120f2",
        "isLock": 0
    }],
    "code": 0,
    "message": "SUCCESS",
    "total": null
}

添加订单

限速规则：/

HTTP请求

GET /user/add_order

参数

memberId 账户ID
direction 订单方向，BUY买，SELL卖
symbol 交易对 BTC/USDT price 价格 amount 数量 type 订单类型，MARKET_PRICE市价，LIMIT_PRICE限价

响应示例

{
    "data": {
        "orderId": 9
    },
    "code": 0,
    "message": "SUCCESS",
    "total": null
}
